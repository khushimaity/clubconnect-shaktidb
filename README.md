# ClubConnect: Digital Campus Community & Event Management Platform using ShaktiDB

## Overview

ClubConnect is a centralized platform designed to manage college clubs, student memberships, and campus events efficiently using ShaktiDB. Students can discover clubs, join communities, register for events, and track their participation history. Club administrators can manage memberships, announcements, event registrations, and attendance records through a unified system.

## Features

* Student Registration and Login
* Club Discovery and Membership Management
* Event Creation and Registration
* Attendance Tracking
* Club Announcements
* Participation History
* Analytics and Reports
